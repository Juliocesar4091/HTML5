How to add images?

In order to add an image we use the tag "img" and the attribute "src"

How to add videos?

In order to add videos we use the tag "video" and the attribute "src"
The text between the <video> and <video> tags will only be displayed in browsers
that do not support the video element or if the source address is wrong or doesn't exist.

PRO TIPS:

-Add thumbnail using the attribute Poster="any image.jpg"

-You can add more than 1 source for the video:
The <video> tag contains one or more <source> tags with different video sources.
The browser will choose the first source it supports or the first one that still exists.
Example: ([]are < >)
[video width="320" height="240" controls] [source src="movie.mp4" type="video/mp4"] [source src="movie.ogg" type="video/ogg"] Your browser does not support the video tag. [/video]
