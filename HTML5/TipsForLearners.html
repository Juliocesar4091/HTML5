Best Tips

Here are some of the things every new Web Developer should know about HTML 5:

-Learn what special Identities are:
To display html symbols like: >,< on your page I recommend you to check out this page W3 Schools Entities
-What are Inline and Block Elemnts:
An inline element cannot contain a block-level element! so to avoid that learn what each tag is Inline & Block Elements
-Never use blank spaces in your file names:
Use underscore sign instead "_" or do something else
-This is an important meta tag to display your webpage correctly on Mobile devices:
Always use <meta name="viewport" content="width=device-width, initial-scale=1.0">.
-Always validate your HTML file:
Always use W3C VALIDATOR to validate your page.
